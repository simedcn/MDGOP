To have a test of MDGOP, directly run main.m where Whitley 5D test function is solved.

MDGOP.m is the main script of MDGOP while all the other functions except for objfun.m are subfunctions called by MDGOP.m.

objfun.m is a small collection of test functions that can be used to test MDGOP.
